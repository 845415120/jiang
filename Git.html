# Git 版本管理

目标： 能够使用 git 管理项目，并且能够将代码上传到Github

首先【Git的基础】 | 猴子都能懂的GIT入门 | 贝格乐（Backlog）open in new window

# 初始化和开发分支

核心步骤：

# 初始化项目仓库
git init -b main

# 将所有文件添加到暂存区
git add .

# 初始化提交
git commit -m '初始化提交'

使用 gitee 管理项目
核心步骤：
在Github上创建项目

  • 本地设置仓库别名
git remote add origin git@github.com:845415120/web-.git
  • 本地推送到远程仓库
# 推送主分支
git push -u origin main
